Implementation of Linear Regression - Car Price Prediction

This project involved implementing a linear regression model to predict car prices based on various features such as mileage, age, brand, etc. The goal was to build a model that could accurately estimate the selling price of used cars, helping both sellers and buyers make informed decisions. The project included data preprocessing, model training, evaluation, and deployment.
